# Critical Rules (絕對守則)
0. **Language**: Always respond in Traditional Chinese (繁體中文).
1. **Audio**: All clicks MUST trigger `playClick()`.
2. **Toast**: All async actions MUST use `showToast()`.
3. **UI**: Dark Cosmic theme (#0f1016), Glassmorphism, Tailwind CSS.
- **Terminology**: Currency is STRICTLY referred to as "Stardust" (星塵). "Coins", "Gold", or "Credits" are forbidden. Storage is "Stardust Vault" (星塵庫).
